This gadget can display an image, static text, a device value (using the {*device*} macro), or any combination —e.g., "The value is {*device*} Celsius."

  • Multi-line text: Supported when "Number of rows" is greater than 0. If the text includes the {*device*} macro, you can choose how many recent values to show.
  • Single-line mode (rows = 0): Text can include valid HTML5 and supports alignment (left, center, right) and styling (bold, italic, underline).
  • Images: Add a background image via the Style tab—images and text can appear together.
Observe that this gadget has 2 texts and both are optional: a "Gadget title" and a "Text (optional)"
