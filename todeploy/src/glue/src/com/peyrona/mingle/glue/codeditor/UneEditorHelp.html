Brief help about the 'Mingle Standard Platform' Editor

All controls in the editor tool bar has a tool-tip with a brief explanation. Just place the mouse cursor over them and it will show up.

Autosave

When checked, files are saved automatically every 5 minutes.


Grid

When checked, files are transpiled for running in a grid.

This value is initially read from file "config.json", but when changed, it prevails over the initial value (file is not altered).


Fake-drivers

If checked, Drivers and Controller would work in faked mode: simulating interaction with the physical device.
This is just a hint: a controller can ignore this request. It depends on whether the controller was created to obey this request or not.

This value is initially read from file "config.json", but when changed, it prevails over the initial value (file is not altered).


Log level

Allows to overwrite the log level sat in the "config.json": this value is initially read from file "config.json", but when changed, it prevails over the initial value (file is not altered).

Different color is applied for SEVERE, WARNING and the rest of levels.


Keyboard shortcuts

Shortcut Action
F1 Shows this help.
Ctrl-Space Launches 'DumbSense' (it is not very 'Intelli' but is better than nothing).
It helps only for drivers (invoke it when cursor is after the DRIVER keyword) and for expressions (invoke it when writing expressions).

Hint: "Templates" combo-box (in editor toolbar) is also a nice help.
Ctrl-N Open an empty editor in a new tab.
Ctrl-O Open file chooser dialog (several file types can be open: Une (source & model), JSON, Java, JS, Python, ...).
Ctrl-S Saves file in current editor.
Ctrl-Shift-S Saves all open files.
Ctrl-T Transpile Une script in current editor (saves the file previously if needed)
Ctrl-R Run script in current editor; saving and transpiling it previously.
You can connect with this running ExEn using the plus button ('+') from the Glue's main too-bar (main window).
Check the assigned Socket port (IP is 'localhost').
Ctrl-F4 Closes current editor.
Ctrl-ArrowUp Duplicates current line in current editor.
Ctrl-ArrowDown Duplicates current line in current editor.
Ctrl-C Copies highlighted text into the clipboard.
Ctrl-V Copies highlighted text from the clipboard into caret position.
Ctrl-Z Undo last typing action.
Ctrl-Y Redo last typing action.
Ctrl-A Select all text.
Ctrl-I Toggle commented estate for selected lines or just current line if no lines are selected.
Ctrl-F Set focus on find (search) text field. If there is any selected text, it is copied here.
Ctrl-Shift-F Copy current editor file name and path to clipboard.
ESC Clears both text fields: search and replace.
